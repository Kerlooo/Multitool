# Multitool

## What is Multitool?
Multitool is a simple hacking tool written in python by me just for fun

# How to install Multitool
## For Debian 12:

```bash
git clone https://github.com/Kerlooo/Multitool
cd Multitool
source myenv/bin/activate
python3 multitool.py
```
